# Kadoma Husband-Gift Quick Guide

呢版係 **現場即睇版**，唔講廢話，目標係去到門真即刻識行。適合鄺太 / 現場決策用。

## 一句總結
**直上 2F outlet floor，先買低風險高命中 gift，鞋最後先睇。**

## 最穩陣購物次序
1. **小皮具 / 配件**
   - wallet
   - card holder
   - belt
2. **袋類**
   - backpack
   - crossbody
   - work bag
3. **男裝上身**
   - polo
   - shirt
   - light jacket
4. **鞋類放最後**
   - 只喺 size 很有把握先買

## 現場行法
### Round 1｜最安全禮物區
- COACH
- TUMI
- Samsonite
- ZERO HALLIBURTON

見到以下就可以認真考慮直接買：
- 黑 / navy / 深啡 wallet
- card holder
- belt
- backpack / business bag

### Round 3｜運動 fallback
- mont-bell

## 最穩陣顏色 / 風格
- **顏色**：black / navy / dark brown / grey
- **風格**：低 logo、乾淨、易襯、唔太 seasonal

## 最好避免
- 太大 logo
- 太鮮色
- 太 trend-driven
- 太貼身嘅衫
- 未確定 size 就買鞋

## 如果只買一份，優先次序
1. wallet
2. card holder
3. belt
4. crossbody bag
5. work bag / backpack

## 90 秒決策法
如果同時見到幾樣都啱：
- 記低 **店名**
- 記低 **item type**
- 記低 **price**
- 記低 **顏色**
- 寫低一句："最體面 / 最實用 / 最抵買"

然後 quick compare 一次再俾錢，唔好無限兜圈。

## Trip execution note
- **先完成 Kadoma 主任務，再諗其他位**
- 唔好一入去就食飯
- 唔好一開始就買鞋
- 公眾假期人流高，**熱門品牌先掃**
