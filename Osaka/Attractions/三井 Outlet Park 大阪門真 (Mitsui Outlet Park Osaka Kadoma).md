# 三井 Outlet Park 大阪門真 (Mitsui Outlet Park Osaka Kadoma)

Day 5 呢個位唔好只當普通 outlet —— 佢其實係**回大阪後最有執行效率嘅 shopping reset day anchor**。同一般「順路行下」唔同，呢度值得有策略，因為你哋前面已經玩咗幾日，體力、袋位、購物優先次序，到呢一日先最清楚。

## ✨ 亮點 (Highlights)
- **Outlet + LaLaport 混合體**：唔係單純 outlet，選擇面比傳統 outlet 更廣。
- **規模夠大**：現有資料約 250 間店，夠你哋做真正集中掃貨。
- **自駕友善**：Day 5 由京都返大阪後直接過去好順，唔使拖行李搭車轉來轉去。
- **適合做旅程中後段採購總結**：前幾日睇中但未買、想補碼、想集中買 gift / 衫 / 波鞋，呢日最啱清。

## 📋 資訊 (Information)
- **地址**：大阪府門真市松生町1-11
- **營業時間**：
  - 商店：10:00 - 21:00
  - 餐廳：11:00 - 21:00
- **交通**：京阪本線 / 大阪單軌電車「門真市站」步行約 8 分鐘
- **停車**：自駕方便，設大型停車場
- **設施特徵**：結合 Mitsui Outlet Park + LaLaport
- **旅客提示**：可先去 1F 服務中心睇外國遊客優惠

## 🧭 Day 5 攻略 (How to Use It Well)
### 1) 呢度唔係朝早慢慢 warm-up 行嘅位，係要 **帶住 shopping mission 入去**
入場前最好已經分好類：
- **必買**：例如波鞋 / 運動衫 / 手袋 / 童裝 / gift
- **可買可唔買**：見到價靚先出手
- **純觀望**：只 check 款 / size / 折扣

如果唔分，250 間店會行到腦都散。呢個唔係浪漫散步位，係購物戰場。😌

### 2) 先掃高優先品牌，再留時間俾「撞彩店」
最錯玩法係一開始就喺細店磨太耐。正路應該：
- 先去你哋最 care 嘅主品牌
- 睇 size / 顏色 / 折扣深度
- 有真價值先落手
- 最後先將剩低時間畀 random browse

### 3) 中段先食，唔好一入去就坐低食飯
11:00 入場，如果一早 lunch，會直接食走黃金購物時間。建議：
- 先衝第一輪 60–90 分鐘
- 真正開始腳軟 / 資訊 overload 先食
- 食飯當 reset，再決定第二輪係補貨定收手

### 4) 呢日最適合買「體積較大但唔急住即刻用」嘅嘢
因為 Day 5 晚上已返大阪酒店，第二日先去機場：
- 可以承受多少少戰利品
- 但都要記住 Day 6 仲有 Rinku，所以唔好第一輪就爆箱

最理想係：
- **Kadoma 買高命中 / 高價值 item**
- **Rinku 買最後補位 / 機場前最後掃尾**

### 5) 4/29 係日本假期氣氛，預人多
Day 5 係 4/29，屬公眾假期氣氛，outlet 人流大機率偏高。最穩陣做法：
- 11:00 左右準時入場
- 熱門品牌先掃
- 唔好假設下晝會更鬆，通常未必

## 🎯 Husband-Gift 攻略（整合 kadoma-shopping-guide）
如果 Day 5 其中一個目標係**幫老公揀禮物**，最有效率玩法應該直接將 **2F outlet floor** 視為主戰場，並按「**低 sizing risk → 高命中率**」順序掃。

### 最穩陣購物次序
1. **小皮具 / 配件**
   - wallet
   - card holder
   - belt
   - key holder
2. **袋類**
   - crossbody bag
   - backpack
   - work bag / briefcase
3. **男裝上身**
   - polo
   - shirt
   - light jacket
   - knit / hoodie
4. **鞋類放最後**
   - 只喺 size / 著法把握高時先出手

### 點解咁排
- 小皮具最少中伏，送禮感又最穩
- 袋類實用、體面、唔太靠身形
- 衫可以好好睇，但對 fit 同 style 要有多少少把握
- 鞋最易買錯，唔應該一開波就衝

### 建議主路線（half-day default）
**Stage 1｜最安全禮物區**
- COACH
- TUMI
- Samsonite
- ZERO HALLIBURTON

先喺呢段搵：wallet / card holder / belt / backpack / work bag。呢幾類通常最容易做到「買得體面又唔易錯」。

**Stage 2｜乾淨男裝區**
- TAKEO KIKUCHI
- TOMMY HILFIGER
- BEAMS OUTLET
- UNITED ARROWS LTD. OUTLET
- Brooks Brothers
- Polo Ralph Lauren Factory Store
- SHIPS OUTLET
- BOSS Outlet / Calvin Klein

呢段啱搵：polo / shirt / light jacket / smart-casual top。

**Stage 3｜運動 / 波鞋 fallback**
- mont-bell
- new balance factory store
- Nike Unite
- adidas Factory Outlet
- ASICS / PUMA
- REGAL / Cole Haan

呢段應該留後。除非一早知 size、或者個人本身真係偏運動 / sneaker / outdoor 路線，否則唔好先衝。

### 老公風格 quick route
- **穩陣送禮型**：COACH → TUMI → Samsonite → TAKEO KIKUCHI
- **office / business casual 型**：TUMI → Brooks Brothers → Polo Ralph Lauren → SHIPS → BOSS
- **年輕少少 / stylish 型**：COACH → BEAMS → UNITED ARROWS → TOMMY → Calvin Klein
- **practical / sporty 型**：mont-bell → UNDER ARMOUR → new balance → Nike → adidas

### 現場揀法小規則
- **最穩陣顏色**：black / navy / dark brown / grey
- **最好揀低 logo、易襯、唔太季節性**
- **避免**：太花、太 loud logo、太貼身、太 trend-driven 嘅 item
- 如果最後只買一份禮物，最穩陣順序通常係：
  1. wallet
  2. card holder
  3. belt
  4. crossbody bag
  5. work bag / backpack

### 時間分配建議
- **首 60–90 分鐘**：COACH / TUMI / Samsonite / ZERO HALLIBURTON
- **之後 60–90 分鐘**：TAKEO / TOMMY / BEAMS / UNITED ARROWS
- **再後 45–60 分鐘**：Brooks Brothers / Polo / SHIPS / BOSS / Calvin Klein
- **最後 30–60 分鐘**：運動 / 波鞋區，只喺仲未買到先去

### 操作小貼士
- 如果同時比較幾樣，可以即場記低：**店名 / item type / price / 顏色 / 點解入圍**
- 門真官方 coupon page 存在，但公開頁面未必直接見到清晰可用優惠，**唔好預設一定有 coupon 救命**；去之前再 check 一次就夠
- 官方亦有提到 **星期日 / 假期 15:00–18:00** 一帶交通擠塞風險較高；雖然你哋 Day 5 未必卡正嗰段，但都值得提早完成主任務

## 💡 Charlotte's Notes
- 呢頁最重要唔係品牌 list，而係**節奏設計**。
- 對成個 trip 嚟講，Kadoma 唔係「其中一個 mall」—— 佢其實係 **Day 5 主任務**。
- 如果當日狀態麻麻，寧願縮減 Senbayashi 時間，都唔好將 Kadoma 行成冇策略亂衝，會白白燒體力。
- 如果 Kadoma 已經成功買到主要戰利品，**後段 Senbayashi 應自動降級成 relax walk / 地道商店街補位**，而唔係再開第二場大型掃貨戰。
